# Keyboard Shortcuts

## Global

| Shortcut | Action |
|---|---|
| Cmd+1 | Switch to Modules mode |
| Cmd+2 | Switch to Racks mode |
| Cmd+3 | Switch to Patches mode |
| Cmd+4 | Switch to Sets mode |
| Cmd+Z | Undo |
| Cmd+Shift+Z | Redo |
| Cmd+= | Zoom in |
| Cmd+- | Zoom out |
| Cmd+0 | Frame to fit |
| Cmd+Shift+L | Toggle labels |
| Cmd+P | Print / export to PDF |

## Modules Mode

| Shortcut | Action |
|---|---|
| Cmd+N | New module from image |
| Cmd+Shift+N | New blank module |
| Cmd+D | Duplicate module (library) / duplicate selected controls (canvas) |
| Delete | Delete selected module(s) (library) / delete selected control(s) (canvas) |
| Cmd+Shift+A | Analyse background image |
| Cmd+A | Select all controls |
| Arrow keys | Nudge selected controls 0.5 pt |
| Shift+Arrow keys | Nudge selected controls 5 pt |

## Racks Mode

| Shortcut | Action |
|---|---|
| Cmd+N | New rack |
| Cmd+Shift+C | New case |
| Cmd+D | Duplicate rack |
| Cmd+A | Select all modules |
| Delete | Remove selected modules or cases |
| Left / Right Arrow | Nudge selected module(s) ±1 HP |

## Patches Mode

| Shortcut | Action |
|---|---|
| Cmd+D | Duplicate patch |
| Delete | Delete selected cables or pin |
| Cmd+Click cable | Insert cable pin at click location |

## Sets Mode

| Shortcut | Action |
|---|---|
| Cmd+N | New set |
| Left Arrow / Cmd+← | Previous patch in set |
| Right Arrow / Cmd+→ | Next patch in set |
